# ibemakil Shop - Интернет-магазин одежды

React-приложение с корзиной, избранным, админ-панелью, API товарами и Local Storage.

## Функции
- Регистрация / Вход (admin@ibemakil.com / admin123)
- Каталог товаров из FakeStore API
- Добавление в корзину / избранное
- Оформление заказа с формой (имя, email, телефон, адрес)
- Админ панель для просмотра заказов
- Данные сохраняются в Local Storage
- Полностью адаптивный дизайн

## Установка
```bash
git clone <your-repo>
npm install
npm start
